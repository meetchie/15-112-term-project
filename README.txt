Project Title and Description
Light: 
The game is a platform based game in which there are three levels you can freely access. 
On the bottom, there will be an underwater cave system, in the middle just caves, and on top, trees. 
The player will have to attempt to traverse the map while finding fireflies. 
Every once in a while, the screen will flash and then an event occurs, 
and the player must seek shelter in the light spot once the flashing occurs. 
Once the player finds all the fireflies, or dies to the spiders when an event occurs, the game ends.

Run Instructions
The file the user should run in an editor is called RUNFILE.py . 
This project is made entirely with cmu_graphics and the PILLOW module, so this library and module should be installed, but there is no data/source files to download.
For further instructions, visit https://www.cs.cmu.edu/~112/notes/tp-related-demos/tp-related-demos.html 

Typical Commands
arrow keys - movement
'c' - cuts spiderwebs
'p' - pause

Shortcut Commands
't' - teleports a user down to effectively skip past a level
'e' - teleports a user halfway down a level
'w' - automatically wins the game
